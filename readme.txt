Flappy Bird 3D
A simple 3D arcade game inspired by Flappy Bird

How to Play
Press SPACE or LEFT CLICK to make the bird jump
Avoid the pipes coming toward you
Pass through gaps to score points

Features
Progressive Difficulty - Pipes move faster as your score increases
Visual Feedback - Bird flashes when scoring

How to Start
Open project in Unity HUB
Open MainMenu scene from Scenes folder
Press Play 
Build Settings: Add MainMenu (index 0) and Game (index 1) scenes (If need)
